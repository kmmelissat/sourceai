# Proceso de Onboarding para Nuevos Comercios B2B

Este es el flujo que el equipo de Customer Success debe seguir una vez que Ventas cierra un contrato:

1. **Recolección de Documentos (KYC):** Solicitar escritura de constitución, NIT de la empresa y DUI/DPI del representante legal.
2. **Creación de Cuenta en Staging:** Se debe habilitar el entorno de pruebas para que el cliente pueda realizar sus primeras transacciones "falsas".
3. **Validación Técnica:** El equipo de ingeniería del cliente debe confirmar que puede recibir nuestros webhooks. (Referenciar documento de arquitectura en `Eng_and_Tech/` si hay dudas).
4. **Paso a Producción:** Una vez validado el cumplimiento (Compliance), se activan las credenciales de producción.

**Métrica Clave:** El tiempo desde la firma hasta la primera transacción real no debe exceder los 5 días hábiles.
